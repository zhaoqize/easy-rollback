# easy-rollback
easy-rollback是一个友好的帮助用户回滚到指定提交的库

# 使用
安装
```js
npm install easy-rollback
```

使用
```js
easy rollback
```

执行 `easy -h`
```js
  Usage: easy [options] [command]

  Options:

    -h, --help  output usage information

  Commands:
    rollback    rollback solution
```
